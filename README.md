# student-performance-analysis-numpy
A complete Student Performance Analysis project built using NumPy, showcasing data loading, statistical analysis, vectorization, broadcasting, sorting, ranking, and performance reporting.
# 📊 Student Performance Analysis using NumPy

## Overview

This project analyzes student performance using **NumPy**. It demonstrates how numerical computing can be applied to real-world educational datasets to generate meaningful insights.

---

## Dataset

- StudentsPerformance.csv
- 1000 student records
- Subjects:
  - Math
  - Reading
  - Writing

---

## Features

- Load CSV dataset using NumPy
- Dataset exploration
- Subject-wise analysis
- Student-wise analysis
- Mean, Maximum, Minimum
- Variance & Standard Deviation
- Boolean Masking
- Grade Generation using `np.where()`
- Broadcasting for Grace Marks
- Pass/Fail Analysis
- Scholarship Eligibility
- Merit List Generation
- Advanced Indexing & Sorting
- Professional Summary Report

---

## NumPy Concepts Used

- Arrays
- Indexing & Slicing
- Boolean Masking
- Aggregation Functions
- Broadcasting
- Vectorization
- `np.where()`
- `np.any()`
- `np.all()`
- `np.argsort()`
- `np.clip()`
